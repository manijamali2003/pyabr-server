# How to install it on Ubuntu server?

 - Git source:
 
 ```shell script
git clone https://github.com/manijamali2003/pyabr-server
 ```

 - Jump to pyabr server directory:
 
 ```shell script
cd pyabr-server
```

 - Create virtual host cloud service:
 
```shell script
sudo python wizard-create.py
```
